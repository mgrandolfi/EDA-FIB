# EDA-FIB

Coursework repository for **EDA (Estructuras de Datos y Algoritmos)** at the **Facultat d'Informàtica de Barcelona (FIB - UPC)**.

This repository contains problem solving material, Jutge exercises, the famous coursework "Game" and coursework related to data structures and algorithms.

## Overview

EDA builds the foundations for reasoning about efficiency, algorithm design, and the selection of appropriate data structures to solve computational problems.

This repo groups together practice work developed throughout the subject, including exercises and project-related material.

## Contents

- Jutge problems
- Game files and players
- Data structure implementations
- Algorithmic exercises
- Course project material

## Tech

- C++
- Problem solving
- Algorithm design
- Fundamental data structures

This is a coursework repository for EDA at FIB.
